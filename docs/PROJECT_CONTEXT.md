# Project Context

## Project
**Star Merchants (SNES)**

Codename: **Project Vibebrew**

Repository: https://github.com/ramgarden/star-merchants-snes

## Overview
Star Merchants is an original SNES homebrew game inspired by the gameplay and atmosphere of the classic BBS game Trade Wars 2002. It is **not** a port and will not use copyrighted assets or code.

The project targets:
- Real SNES hardware
- Accurate emulators
- PVSnesLib

## Current Status
- Phase 0 complete
- Build 0001 boots successfully
- First custom ROM displays:
  - STAR MERCHANTS
  - PROJECT VIBEBREW
  - BUILD 0001

## Bootstrap
The game project was bootstrapped from the official `hello_world` example included with PVSnesLib, then customized.
